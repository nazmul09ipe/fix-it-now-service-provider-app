# 🔧 Fix It Now - Home Service Booking Platform

A modern and responsive home service booking platform where users can browse services, view service details, book services, manage bookings, and connect with service providers.

## 🌐 Live Site

🔗 https://fix-it-now-app.web.app

---

## 📌 Project Overview

Fix It Now is a full-stack service marketplace designed to connect customers with skilled service providers. Users can explore available services, book appointments, and manage service requests through a secure and user-friendly interface.

---

### 👤 Authentication & Authorization
- Firebase Authentication

- Browse all available services

### 📅 Booking System
- Book services online
- Track booking status

## 🔐 Environment Variables

Create a `.env.local` file in the root directory:

```env
VITE_API_URL=https://your-server-url.vercel.app

VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/nazmul09ipe/fix-it-now-service-app-client-side.git
```

### Navigate to Project

```bash
cd fix-it-now-service-app-client-side
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
